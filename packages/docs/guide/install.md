# 安装

只支持可运行web comopnent的浏览器

## 使用包管理器

::: code-group

```sh [npm]
npm install @lingjhf/guava
```

```sh [pnpm]
pnpm add @lingjhf/guava
```

```sh [yarn]
yarn add @lingjhf/guava
```

:::

Vue3推荐安装

::: code-group

```sh [npm]
npm install @lingjhf/guava-vue
```

```sh [pnpm]
pnpm add @lingjhf/guava-vue
```

```sh [yarn]
yarn add @lingjhf/guava-vue
```

:::
